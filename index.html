Technical Skills Software and Lanuages

I have had experience using both Python and Java 8 when programming, with my most confident being Python. With these I have used the Eclipse IDE and Visual Studio Code. I will also be looking at both Scheme and Haskell during my second year. I have also been looking at website development over my summer and taught myself the basics of HTML, CSS and the CSS framework Bootstrap 4 to create this cv/portfolio website. My primary OS is Windows 10, however I also have experience using both Ubuntu and ElementaryOS Linux distributions. I have used these for both programming and taking part in cyber security related capture the flag competitions.

I am a frequent user of the Microsoft Office suite and I am confident using the applications, especially Word and Excel. As well as this, I am also a user of the Adobe Creative Cloud and I have taught myself to use both Photoshop and Illustrator. Over the past few years, I have used both of these to complete graphics commisions for gaming Youtube and Twitch channels.
